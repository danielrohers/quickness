## Quick Start

The quickest way to get started with quickness is to utilize the executable `express(1)` to generate an application as shown below:

Create the app:
```bash
$ npm install -g quickness
$ quickness foo && cd foo
```

Install dependencies:
```bash
$ npm install
```
or
```bash
$ yarn
```

Development
```bash
$ npm start
```
